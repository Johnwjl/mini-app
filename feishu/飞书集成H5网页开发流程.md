# ⻜书集成之自建应用使用流程 

（2022-03-22）

## 一. ⻜书桌面客户端 - 工作台

- 点击 【创建应用】，将进入 【⻜书开放平台】网⻚端

![image-20220322231512660](http://gitee.com/johnwjl/my_picgo/raw/master/image-20220322231512660.png)

## 二. ⻜书开放平台

- 点击 【创建应用】- 将进入 【我的后台】

- 【我的后台】- 点击 【创建企业自建应用】

- 填写 【应用名称】、【应用描述】，确定创建

- 进入创建的应用 - 【成员管理】，添加该应用的管理员与开发人员的权限
